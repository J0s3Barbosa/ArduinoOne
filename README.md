# ArduinoOne

* html, css, javascript interface to control arduino robot thru Jquery
* Send data to an APi where arduino is connected 

* Repository

https://github.com/appchto/ArduinoOne 


### Version: 0.0.1

### API Usage

```sh
npm install

npm start

# Or run with Nodemon
npm run dev

# Visit http://localhost:8080

```
